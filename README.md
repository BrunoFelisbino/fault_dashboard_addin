# fault_dashboard_addin

Add-in Geotab para dashboard de falhas da frota.

## Arquivos

- `fault_dashboard.html`
- `fault_dashboard_addin_config.json`
- `fault_dashboard_icon.svg`

## Instalação

1. Publique estes arquivos no GitHub Pages.
2. Use a URL do `fault_dashboard_addin_config.json` no cadastro do add-in no Geotab.
3. Mantenha `fault_dashboard.html` e o ícone acessíveis publicamente.

## Observação

- O idioma principal está em `pt-BR`.
- A versão atual do pacote é `3.1.1`.
